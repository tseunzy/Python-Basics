**Personal Finance Tracker**

**Features:**
    Add income and expense transactions
    Categorize expenses
    View total balance
    Analyze spending by category

**Number Guessing Game**
A fun game where the computer selects a random number and the user tries to guess it.

**Features:**
    Random number generation
    User input and feedback
    Hints for high or low guesses
    Tracks number of attempts


**Multiplication Table Generator**
Generates multiplication tables for a given number.

**Features:**
    User-defined number and range
    Neatly formatted output

**Star Pyramid Pattern**
Prints different star (*) patterns using loops.

**Features:**
    Pyramid and triangle star patterns
    Adjustable height

**Temperature Converter**
Converts temperature between different units.

**Conversions Supported:**
    Celsius to Fahrenheit
    Fahrenheit to Celsius
    Celsius to Kelvin
    Kelvin to Celsius
